# fishmart.ru

## setting up
```
mkdir code; \
cd code; \
git clone https://github.com/jintaxi/fishmart.ru.git; \
cd fishmart.ru; \
python3.9 -m venv fishmart.ru; \
source fishmart.ru/bin/activate; \
pip install -U pip; \
pip install -r equirements.txt
```
